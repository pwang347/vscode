# VS Code Mobile — Capacitor Project

This directory contains the Capacitor v6 project for the VS Code Mobile app.

## Prerequisites

- Node.js 18+
- Android Studio (for Android development)
- Xcode 15+ (for iOS development, macOS only)
- Tailscale installed on both the mobile device and the VS Code server machine

## Development

### First-time setup

```bash
cd mobile
npm install
```

### Build the web bundle

```bash
npm run build:web
```

### Sync web assets to native projects

```bash
npm run cap:sync
```

### Run on device/emulator

```bash
# Android with live reload
npm run cap:run:android

# iOS with live reload (macOS only)
npm run cap:run:ios

# Open in Android Studio
npm run cap:open:android

# Open in Xcode
npm run cap:open:ios
```

## Architecture

The Capacitor app wraps the VS Code mobile web build and provides:

- **Native WebView**: Renders the mobile workbench UI
- **Capacitor Plugins**: Bridge to native capabilities (haptics, push notifications, etc.)
- **Tailscale Integration**: Network-level VPN (installed separately on the device)

The web content is built from `src/vs/mobile/` using the mobile-specific entry points
and served from `out-vscode-mobile/`.

## Directory Structure

```
mobile/
├── capacitor.config.ts   # Capacitor configuration
├── package.json           # Mobile-specific dependencies
├── android/               # Android project (generated by `npx cap add android`)
├── ios/                   # iOS project (generated by `npx cap add ios`)
└── resources/             # App icons, splash screens
```

The `android/` and `ios/` directories are generated by Capacitor CLI:

```bash
npx cap add android
npx cap add ios
```
